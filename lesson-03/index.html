Git

  1. Створити репозиторій "beetroot" на github c домашніми роботами
  2. В корні репозиторія створити файл index.html
  3. Опублікувати репозиторій 
          git init
      git add .
      git commit -m 'first commit'
      git remote add origin https://github.com:*your accout github*/*your repository name*.git
      git push -u origin master
  4. В репозиторії "beetroot" створити гілку "gh-pages" 
          git checkout -b gh-pages
  5. Опублікувати гілку 
          git push -u origin gh-pages
  6. Не забути переключитися назад на гілку мастер 
          git checkout master
  7. Створити новий репозиторій з іменем *your account name*.github.io в який помістити свож резюме (аналогічно кроку 3)
