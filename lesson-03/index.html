Git

  1. Створити репозиторій "beetroot" на github c домашніми роботами
  2. В корні репозиторія створити файл index.html
  3. Опублікувати репозиторій 
          git init
      git add .
      git commit -m 'first commit'
      git remote add origin https://github.com:*your accout github*/*your repository name*.git
      git push -u origin master
  4. В репозиторії "beetroot" в налаштуваннях увімкнути "Github Pages" на галці "master"
  5. Створити новий репозиторій з іменем *your account name*.github.io (наприклад "a1exalexander.github.io") в який помістити свож резюме (аналогічно кроку 3)
